What would it be like if the sites of the 90's and early 2000's intersected with the tech of today? How would social media work with the technology of yesteryear? Well look no further. This template can be modified to use API's but also reuse older networking protocols.

So breakout that old 486DX33, and relive these early days where the Terminal was the only thing that was there.

This site has been tested with terminal based browsers such as lynx, w3m, links and browsh.

SPECIAL NOTE: Some terminal based browsers need a little tweaking in order to activate the URL schemes (Such as telnet and IRC).
